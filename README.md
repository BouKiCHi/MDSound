# MDSound
メガドライブ サウンドチップエミュレーション.DLL
  
[概要]  
 このDLLは、VGM Playerのソースからメガドライブに搭載されている以下の音源チップの動作をC#向けコードに移植したものです。  
    YM2612 , SN76489 , RF5C164 , PWM  
  
[機能、特徴]  
 ・割と.NETの文化に沿った記述が可能です。  
 ・マネージドなプログラムです。(testプログラムはSDLNETを使用していますが。)
